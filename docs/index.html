Rejstříky ČR

Agent pro české veřejné rejstříky — ARES, Registr smluv a insolvenční řízení. Odpovídá česky, data čerpá výhradně z API (nic si nevymýšlí).

Co agent umí

  • Vyhledat firmy podle názvu, IČO, města nebo ulice
  • Zobrazit kompletní detail firmy (sídlo, právní forma, DIČ, CZ-NACE, registrace…)
  • Vypsat statutární orgány firmy (jednatele, členy představenstva, společníky)
  • Zobrazit živnostenská oprávnění firmy
  • Ověřit insolvenci firmy (rychlá kontrola přes CEÚ)
  • Vyhledat smlouvy v Registru smluv (podle IČO nebo textu)
  • Zobrazit detail smlouvy včetně příloh a problémů
  • Vyhledat insolvenční řízení (detailní, přes Hlídač státu)
  • Zobrazit detail insolvenčního řízení (dlužníci, správci, dokumenty)
  • Validovat českou adresu proti RÚIAN
  • Kombinovat nástroje — „prověř firmu" = detail + insolvence + jednatelé + smlouvy

Co agent neumí / omezení

  • Hledat fyzické osoby (pouze firmy s IČO)
  • Zobrazit sbírku listin / výroční zprávy
  • Zobrazit data z katastru nemovitostí
  • Odpovídat na dotazy mimo české rejstříky (odmítne)
  • IČO kratší než 8 číslic — agent by měl doplnit nuly, ale občas je zapomene

Dostupné nástroje

NástrojZdrojPopis
searchCompanies ARES Vyhledání firem dle názvu, IČO, města, ulice (max 100 výsledků)
getCompanyDetail ARES Kompletní profil firmy podle IČO
getCompanyOfficers ARES Statutární orgány z Obchodního rejstříku
getTradeLicenses ARES Živnostenská oprávnění (RŽP)
checkInsolvency ARES (CEÚ) Rychlá kontrola insolvence — ano/ne
validateAddress ARES (RÚIAN) Ověření adresy proti registru adres
searchContracts Hlídač státu Hledání smluv v Registru smluv (25 na stránku, stránkování)
getContractDetail Hlídač státu Detail smlouvy — strany, částky, přílohy, problémy
searchInsolvencyCases Hlídač státu Hledání insolvenčních řízení (detailnější než CEÚ)
getInsolvencyDetail Hlídač státu Detail insolvence — dlužníci, správci, věřitelé, dokumenty

Dva zdroje insolvencí: Rychlá kontrola checkInsolvency (ARES/CEÚ) vrací jen ano/ne. Pro detailní informace o řízení, dlužnících a dokumentech použijte Hlídač státu nástroje (searchInsolvencyCases / getInsolvencyDetail).

Známé problémy a omezení

Pomalé odpovědi u kombinovaných dotazů

Dotaz typu „prověř firmu" volá 4 API najednou (detail + insolvence + jednatelé + smlouvy). Odpověď může trvat 15–30 sekund. Buďte trpěliví.

Hlídač státu — stránkování

Výsledky smluv i insolvencí jsou stránkovány po 25. Agent by měl nabídnout „další stránku", ale ne vždy to udělá automaticky.

IČO vždy 8 číslic

IČO musí mít 8 číslic. Pokud zadáte kratší (např. „123456"), agent by měl doplnit nuly na „00123456". Pokud ne, zadejte celé číslo.

Často kladené otázky

Odpovědi na časté dotazy ohledně agenta Rejstříky ČR.

Kde agenta najdu v M365 Copilotu?

Agent je dostupný výhradně v prostředí M365 Copilot (nikoliv v Teams). Najdete ho v levém panelu Copilotu, nebo ho zavolejte přes ikonu @ — začněte psát @Rejstříky ČR.

Po zmínce agenta můžete rovnou psát svůj dotaz.

Agent neodpovídá / vrací chybu

Možné příčiny:

  • Backend se probouzí — po delší neaktivitě může server potřebovat pár sekund na start.
  • API třetí strany nefunguje — ARES nebo Hlídač státu API může být dočasně nedostupné.
  • Neplatný dotaz — zkuste přeformulovat nebo zadat konkrétní IČO / název firmy.

Zkuste dotaz zopakovat za minutu. Pokud problém přetrvává, kontaktujte správce.

Agent si vymýšlí data

Agent je nastavený tak, aby nepoužíval vlastní znalosti a čerpal pouze z API. Pokud si přesto vymýšlí:

  • Ověřte, že jste agenta zavolali přes @Rejstříky ČR (ne obecný Copilot)
  • Zkuste dotaz přeformulovat — někdy LLM „přeskočí" volání API
  • Pokud se to děje opakovaně, nahlaste to
Jak poznám, že agent zavolal API?

V odpovědi byste měli vidět strukturovaná data (tabulky, odrážky s konkrétními IČO, adresami, čísly). Pokud odpověď vypadá obecně a vágně, agent pravděpodobně API nezavolal.

U dat z Hlídače státu by se na konci měl objevit odkaz na zdroj.

Jak hledat smlouvy?

Nejspolehlivější je hledat podle IČO:

  • „Najdi smlouvy pro IČO 45274649"
  • Můžete hledat i textově: „Najdi smlouvy na dodávku elektřiny"
  • Pro další stránku řekněte: „Ukaž další stránku výsledků"
Co znamená „prověř firmu"?

Kombinovaný dotaz, který zavolá 4 nástroje najednou:

  1. getCompanyDetail — základní profil
  2. checkInsolvency — kontrola insolvence
  3. getCompanyOfficers — jednatelé / představenstvo
  4. searchContracts — smlouvy v Registru smluv

Odpověď může trvat déle (15–30 s).
